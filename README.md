# EKG Denoising App

A basic React Native and Expo mobile prototype for viewing mock denoised ECG data.

## Features

- Expo SDK 54 app using the same core versions as `Shelfie-App`
- Mock clean, noisy, and denoised ECG signal data
- SVG-based ECG charts for mobile screens
- Normal ECG and arrhythmia comparison examples

## Scripts

```bash
npm start
npm run android
npm run ios
npm run lint
```

## Core Versions

- Expo `~54.0.33`
- React `19.1.0`
- React Native `0.81.5`
